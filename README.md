# AirBnb clone

## Technologies

- [SvelteKit](https://kit.svelte.dev/)
- [Kahi UI](https://kahi-ui.nbn.dev/)
- [Clerk](https://clerk.dev/)
- [Rowy](https://www.rowy.io/)

## Install

```bash
# clone git repository
git clone https://github.com/samullman/airbnb.git

# install dependencies
yarn

# start server
yarn dev
```
